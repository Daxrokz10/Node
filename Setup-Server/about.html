About This Node Server

This server is a simple example built with Node.js to demonstrate serving static HTML pages using the built-in http and fs modules. The UI is styled with Bootstrap 5 for a modern look.

  • Serves static pages: Home, About, Contact
  • Handles 404 errors with a custom page
  • Great for learning the basics of Node.js servers
